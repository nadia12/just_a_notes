Starting with Upstart (Ubuntu 14.04)

1. Create CONF file on etc/init
2. Configure the runlevels
3. Check to avoid issues 'init-checkconf /etc/init/<filename>.conf'
4. 'sudo service filename <action>' (from file conf)
